== README

Tahanan ni Aling Meding Hotel is my mom's and her 10 siblings' mansion in Laguna, San Pablo Philippines. 
This is Tahanan's original website built in 2010: http://www.tahananhotel.com/
I built this as a personal project, to give the old site a new look and to improve my front-end skills.

Technologies used:

* Ruby version 2.0.0

* Rails 4.0.4

* ProstgreSQL

* HTML5

* CSS3/SCSS

* Bootstrap

* JavaScript

* JQuery (Jssor Slider, Magnific Popup)

* Google Maps API v3
